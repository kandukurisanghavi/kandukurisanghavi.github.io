🥗 Food Calorie Tracker App

This full-stack web app empowers users to track calories based on meals, exercise, and nutrition data. Built with Django and integrated with an external Nutrition API, the app offers real-time food analysis, graphs, and personalized insights to support healthy lifestyle choices.

📌 Problem Statement

People often struggle to understand the nutritional value of their food and track calories effectively. Manual tracking is time-consuming and inaccurate, and most solutions lack integration with real-time data or personalized goals.

✅ Features Implemented

  • 🔐 Credential validation and user authentication
  • 🍎 Food search and calorie data using Nutrition API
  • 📋 Favorite food list for quick access
  • 📉 Calorie tracking for both meals and exercise
  • 📊 Nutrition data visualized with interactive graphs
  • 🗑️ Delete confirmation pop-up for better UX

🛠️ Technical Stack

  • Backend: Django Framework (Python)
  • Frontend: HTML, CSS (Bootstrap)
  • API: Nutritionix or similar external API
  • Database: SQLite
  • Visualization: Chart.js (for nutrition graphs)

📚 Project Learnings

  • Integrated and consumed third-party APIs
  • Designed Django models aligned with ER diagrams
  • Handled user input securely and efficiently
  • Improved UX with form validation and modals
